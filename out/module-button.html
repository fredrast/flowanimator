Module: button

Defines the Button class, which generates buttons on the SVG canvas, draws their icons and defines their hover and click behavior. Idea and code example from https://codepen.io/fuzzyma/pen/wQbVed Could be better to create HTML buttons instead, that would enable the buttons to receive focus when pressing the tab key, and that would also make it easier to create a fully responsive layout.
