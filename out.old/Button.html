Class: Button

Button(type, canvas, x, y, radius, clickHandler)

new Button(type, canvas, x, y, radius, clickHandler)

Generates buttons on the SVG canvas, draws their icons and defines their hover and click behavior. Idea and code example from https://codepen.io/fuzzyma/pen/wQbVed Could be better to create HTML buttons instead, that would enable the buttons to receive focus when pressing the tab key, and that would also make it easier to create a fully responsive layout.

Parameters:
Name Type Description
type

Defines the type of button to be created, or more precisely, the type of icon to be drawn on the button. Following types supported: open, play, stop, slower and faster.
canvas

A reference to the SVG canvas on which the button should be drawn.
x

x coordinate of the button on the canvas
y

y coordinate of the button on the canvas
radius

The radius of the button
clickHandler

Click handling function to be set on the button
